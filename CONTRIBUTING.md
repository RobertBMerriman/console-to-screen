# Contributing

## Tech

React (Vite) + React Compiler, TypeScript, shadcn/ui + tailwind, zustand, Tanstack Form, zod

## Run locally

Everything is just in the React SPA so it's just the classic:

```
pnpm install
pnpm dev
```
